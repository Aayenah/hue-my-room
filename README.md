## Introduction
A bot that allows Twitch chat to control my Phillips Hue lights. 

## List of Commands
- `!on` - turns lights on.
- `!off` - turns lights off.
- `!{colourName}` - changes light colour to one of the available colours (e.g. `!red`, `!green`, `!pink`).
- `!satUp` - increases colour saturation by 20 (max 254).
- `!satDown` - decreases colour saturation by 20 (min 0).
- `!hue` - a help command that points chat user to list of commands.
